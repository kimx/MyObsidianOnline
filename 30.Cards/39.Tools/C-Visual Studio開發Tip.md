## 快捷指令
- 註解選取行 : Ctrl + K, Ctrl + C
- 上面插入一個空行 :  Ctrl + Enter
- 下面插入一個空行 :  Ctrl + Shift + Enter
- 選取目前文字 : Ctrl + W
- 剪切當前行 :	Ctrl + L
- 删除當前行： Ctrl + Shift + L
- 顯示參數資訊: Ctrl + Shift + 空格
- 顯示物件的成員清單 : Ctrl + J
- 展開/收合: Ctrl + M,M
